# StudyKit-forms
Studying Tool Made For Memorizing Words
<br>
Made in c# forms
<br>
[![MIT License](https://img.shields.io/badge/License-MIT-purple.svg)](https://choosealicense.com/licenses/mit/)

## Features
* Prompt System
* Save/Load via Json
* Checking for errors
* Whitespace proof environment
